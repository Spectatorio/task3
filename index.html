Terminal-based text editor in C++

Developed a terminal-based text editor in C++ utilizing object-oriented programming and modular code organization. Designed editor using a model-view-controller architecture pattern, separating concerns into buffer model, rendering view, and input handling controller. Participated in adding features like search/replace using KMP string matching algorithm, and markdown syntax highlighting.

  • Technologies: C++, MVC Architecture
  • Features: Search/Replace, Syntax Highlighting
  • Category: Software Development

  • Architecture: MVVM, Repository Pattern
  • Category: Mobile Development
